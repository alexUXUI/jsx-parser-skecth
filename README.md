# jsx-parser-skecth

## Up and running

1. Clone the repo

```shell
$ git clone https://github.com/alexUXUI/jsx-parser-skecth.git
```

2. Install dependencies

```shell
$ cd jsx-parser-skecth
```

3. Start the dev server

```shell
$ yarn && yarn dev
```

4. Visit the app
   go to http://localhost:8080
